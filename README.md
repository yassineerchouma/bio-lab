🧬 BioTech Lab Portal
A modern, responsive web application designed for genomic analysis. BioTech Lab allows users to decode DNA sequences into proteins, perform NCBI BLAST searches to identify genetic matches, and generate professional PDF reports of their findings.

✨ Key Features
DNA to Protein Translation: Instantly convert nucleotide sequences (ATGC) into amino acid chains.

NCBI BLAST Integration: Connects to the National Center for Biotechnology Information (NCBI) database to find top sequence matches.

Smart Caching: Optimized performance using a backend cache to reduce redundant API calls and save server resources.

Exportable Reports: Generate and download PDF summaries of your genomic analysis.

Modern UI/UX: * Fully responsive grid layout for mobile and desktop.

Interactive Dark Mode with persistent user preferences.

Dynamic DNA helix background animations.

Local History tracking (Top 5 searches).

🚀 Tech Stack
Frontend: HTML5, CSS3 (Custom Grid/Animations), JavaScript (ES6+).

Backend: Python, Flask.

Bioinformatics: NCBI BLAST API, Biopython (or custom parsing logic).

Deployment: PythonAnywhere.
